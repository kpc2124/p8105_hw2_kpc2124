Read and clean precipitation data for 2017 and 2018, combine them

``` r
precip_2017_df = 
  read_excel("./data/HealthyHarborWaterWheelTotals2018-7-28.xlsx", range = "2017 Precipitation!A2:B14") %>% 
  janitor::clean_names() %>%
  mutate(year = 2017)

precip_2018_df = 
  read_excel("./data/HealthyHarborWaterWheelTotals2018-7-28.xlsx",range = "2018 Precipitation!A2:B9") %>% 
  janitor::clean_names() %>%
  mutate(year = 2018)

precip_tidy = 
  bind_rows(precip_2017_df, precip_2018_df) %>% 
  janitor::clean_names() %>% 
  mutate(month = month.name[month]) %>%
  select(month, year, total)
```

variables in this dataset are: dumpster, month, year, date,

cigarette\_butts, glass\_bottles, grocery\_bags, chip\_bags,

dumpster in 2017 is 8. The size of the Precipitation dataset is 57
observations. The precipitation dataset contains the variables: month,
year, total. The total precipitation in 2018 is 23.5.
